### Ручной способ

После мерджа вашего PR в ```master```, вам необходимо:

1. Перейти в ветку ```master```
2. Обновить версию пакета с помощью ```npm version patch|minor|major```
3. Выполнить команду ```git push origin master --follow-tags```
